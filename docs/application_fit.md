# Application Fit

## Projektziel

Der Java Insurance Claims Manager simuliert einen vereinfachten Versicherungsprozess. Er zeigt Java/OOP, CSV-Verarbeitung, Validierung, Risiko-Scoring, Services, Reports und JUnit-Tests.

| Generali | Versicherungsprozesse und Datenanalyse | CSV, Validierung, Reporting | Als fachliches Java-Projekt präsentieren |

## Level-Einschätzung

Level 3: starkes Bewerberprojekt. Das Projekt zeigt Java/OOP, Tests, CSV-Daten, Validierung und fachlichen Bezug ohne unnötige Komplexität.
