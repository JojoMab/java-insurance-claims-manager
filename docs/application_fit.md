# Application Fit

## Projektziel

Java Insurance Claims Manager simuliert einen vereinfachten Versicherungsprozess mit synthetischen Schadensfällen. Das Projekt zeigt Java/OOP, CSV-Verarbeitung, Risiko-Scoring, Plausibilitätsprüfung, Reporting und JUnit-Tests auf Bewerberniveau.

## Skill-Abdeckung

- Java 21
- objektorientierte Modellierung
- Enums für ClaimType und ClaimStatus
- CSV-Verarbeitung
- Risiko-Score von 1 bis 10
- Plausibilitätsprüfung
- Service-Schicht
- Report-Erstellung
- JUnit 5
- Maven
- GitHub Actions CI

## Level-Einschätzung

**Level 3 – Starkes Bewerberprojekt**, wenn Tests und CI erfolgreich laufen.

Begründung: Das Projekt zeigt ein klares fachliches Szenario, Java/OOP, Services, Validierung, Risiko-Logik, synthetische Daten, Reports und Tests. Es bleibt bewusst auf Bewerberniveau und behauptet kein echtes Versicherungssystem.
