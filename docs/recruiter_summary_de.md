# Recruiter Summary – Java Insurance Claims Manager

## Kurzbeschreibung

Java Insurance Claims Manager ist ein Bewerberprojekt für duale Studiengänge in Informatik und Wirtschaftsinformatik. Das Projekt simuliert einen vereinfachten Versicherungsprozess mit synthetischen Schadensfällen, CSV-Verarbeitung, Plausibilitätsprüfung, Risiko-Score und Report-Erstellung.

## Was das Projekt zeigt

- Java 21 und objektorientierte Modellierung
- klare Trennung von Model, Service, IO und Utility-Klassen
- Enums für Schadensart und Bearbeitungsstatus
- CSV-Verarbeitung mit synthetischen Beispieldaten
- Risiko-Score von 1 bis 10
- Prioritätslogik für Schadensfälle
- Plausibilitätsprüfung von Pflichtfeldern und Beträgen
- Report-Erstellung als Textdatei
- JUnit-Tests für Risiko-, Service- und Validierungslogik
- GitHub Actions als CI-Grundlage

## Bewerbungsbezug

Das Projekt eignet sich besonders für Bewerbungen bei Versicherungen, Finanz-IT-Unternehmen und wirtschaftsinformatiknahen Arbeitgebern. Es passt fachlich zu Allianz, Generali, Atruvia, Finanz Informatik, Bayerische Versorgungskammer, BG-Phoenics und ConSol.

## Wichtiger Hinweis

Alle Daten sind synthetisch. Das Projekt ist ein Bewerberprojekt und bildet kein echtes Versicherungssystem ab.
